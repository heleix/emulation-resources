How to Use This Guide

This is an asynchronous lesson. Following along with the entirety of this lesson is designed to take up 45-60 minutes. The guide is divided into three separate modules estimated to take 15-20 minutes each. However, you may return to specific modules or spend more time with others if needed.

While this guide is designed for beginners, we expect you to have a basic understanding of computing and information technologies, e.g. you should be able to understand what an operating system is and explain elementary differences between hardware, software, and digital media formats.


Learning Objectives

By completing this lesson, you should be able to:

- Acquire a basic understanding of emulation technology
- Consider how emulation would be useful for archivists
- Become aware of various considerations for implementing emulation as an access strategy
- Use emulation to solve an example access problem
